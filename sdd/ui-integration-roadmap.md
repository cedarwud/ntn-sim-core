# UI Integration Roadmap (Outline)

**Status:** Outline only  
**Blocked by:** Phase 0 through Phase 5

## Later Goal

Expose simulator functionality through a stable, provenance-aware UI without leaking internal runtime details.

## Current Approved Scope

1. baseline reproduction viewer
2. baseline KPI / result chart presentation
3. exposure alignment and existing UI/UX improvements

## Deferred Scope

1. MODQN comparison dashboards
2. HOBS / EE variant comparison views
3. estnet-driven consumer integration

## Near-Term Scope

1. Realistic / Advanced / Sensitivity / Internal-only exposure mapping
2. parameter-aware controls and presets
3. derived-quantity display rules
4. baseline result / replay / figure navigation

## Reason for Deferral

UI cannot be stabilized before parameter registry, model bundles, and runtime contracts are stable.
